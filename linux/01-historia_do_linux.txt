            história do linux

linux é um sistema operacional de código aberto tipo unix (s.o. da empresa de telecomunicações AT&T)
    AT&T licenciou o código-fonte para universidades por um valor simbólico

1984 => richard m. stallman inicia o projeto GNU
    objetivo de reescrever o unix
    ainda necessitava do kernel
    surge a licença GPL => GNU Public License
        GPL possui algumas características:
            softwares produzidos sob essa licença devem ser distribuídos livremente
            pessoas com acesso ao código podem utilizar, editar e criar softwares a partir dele
            softwares criados podem ser revendidos, porém mantidos como livre

1991 => Linus Torvalds começa a desenvolver seu próprio kernel
    objetivo era usar em casa o mesmo s.o. que usava na universidade (unix)
    este kernel era o que faltava para a criação de um s.o. tipo unix com licença gpl

a partir do kernel linux, as distribuições começaram a ser montadas (Red Hat, Debian, Suse...)
algumas distribuições se referem ao kernel como GNU/linux
